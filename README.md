# Development Team Project: Coding Output

# Prerequisite

The following software required.
- Docker

# Docker
Deploy containers
```
docker compose up --build -d
```
Remove
```
docker-compose down
```
# Program

publisher.py: Publish Hello messages every 2 sec
subscriber.py: Subscribe topic every sec

# Console page
Solace: http://localhost:8080/
Solace-exporter: http://localhost:9628/
Prometheus: http://localhost:9628/
Grafana: http://localhost:3000/
